# Electricity Bill Management System

## Project Overview
This project is a centralized database solution designed to modernize traditional electricity billing procedures by replacing manual data entry with an automated system.

## Goals and Objectives
**Centralize Data Management**: Consolidating consumer, meter, and billing history.
**Real-Time Monitoring**: Tracking electricity distribution and consumption.
**Error Reduction**: Minimizing human entry errors through automated processes.

## Database Schema
The system uses six core tables:
1. **User**: Personal profiles and contact information.
2. **Meter**: Hardware tracking and installation details.
3. **UsageData**: Records of kWh consumption with timestamps.
4. **BillingInfo**: Invoices generated based on specific usage data.
5. **PaymentTransaction**: Real-time status of payment attempts.
6. **Receipt**: Finalized payment confirmations.

## Future Improvements
Implementation of Smart Meters (AMR) for remote reading.
Enhanced security protocols and offline mobile accessibility.
