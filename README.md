# Electricity Bill Management System

## Project Overview
[cite_start]This project is a centralized database solution designed to modernize traditional electricity billing procedures by replacing manual data entry with an automated system[cite: 4].

## Goals and Objectives
* [cite_start]**Centralize Data Management**: Consolidating consumer, meter, and billing history[cite: 4].
* [cite_start]**Real-Time Monitoring**: Tracking electricity distribution and consumption[cite: 4].
* [cite_start]**Error Reduction**: Minimizing human entry errors through automated processes[cite: 4].

## Database Schema
[cite_start]The system uses six core tables[cite: 4]:
1. **User**: Personal profiles and contact information.
2. **Meter**: Hardware tracking and installation details.
3. **UsageData**: Records of kWh consumption with timestamps.
4. **BillingInfo**: Invoices generated based on specific usage data.
5. **PaymentTransaction**: Real-time status of payment attempts.
6. **Receipt**: Finalized payment confirmations.
